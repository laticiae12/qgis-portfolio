# QGIS Geospatial Portfolio – Laticia Edwards

## Overview
This portfolio demonstrates geospatial analysis skills using QGIS, focused on healthcare access and logistics optimization.

## Project 1: Dialysis Clinic Access Map (Dallas, TX)

### Objective
Analyze access to dialysis clinics and identify underserved areas.

### Data Used
- Dialysis clinic locations
- Population density (census data)
- Road network data

### Analysis Performed
- Buffer analysis (5–10 mile radius from clinics)
- Heatmap of patient density
- Spatial overlay to identify gaps in coverage

### Outcome
Identified areas with limited access to dialysis care and opportunities for improved medical courier routing.

## Tools Used
- QGIS
- OpenStreetMap
- Census Data
